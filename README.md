# Twitter Sentiment Classification

## 📌 Overview

This project builds a Machine Learning model to classify tweets into sentiment categories using Natural Language Processing (NLP).

It demonstrates an end-to-end workflow including data cleaning, feature engineering, model training, and evaluation.

---

## 🎯 Objectives

* Clean and preprocess raw tweet text
* Convert text data into numerical features using TF-IDF
* Train a classification model
* Evaluate model performance

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

---

## ⚙️ Project Workflow

1. Data Cleaning (remove URLs, special characters, lowercase text)
2. Feature Extraction using TF-IDF
3. Train/Test Split
4. Model Training (Logistic Regression)
5. Model Evaluation
